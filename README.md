## 💻 Getting Started <a name="getting-started"></a>

> To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

Windows, Mac, Linux

- Download and install a Ruby on your computer.
- Download and install a Node on your computer.

### Setup

Clone this repository to your desired folder by running the command below in your terminal:

```
git git@github.com:Fayob/TDD-RSpec.git
```

- Navigate to the directory
- Run the code from the file

### Run tests

To run tests, run the following command:

```sh
  rspec spec  solver_spec.rb
```
